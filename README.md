# Projet DHT11 + Écran LCD

Ce projet mesure la température et l’humidité avec un capteur DHT11.
Les résultats s’affichent sur un écran LCD 16x2 via une carte Arduino.

Pour le faire, il faut :
- Un capteur DHT11.
- Un écran LCD 16x2 avec module I2C.
- Une résistance à 5K ohm.
- Une carte Arduino.
- Une breadboard et des fils.
